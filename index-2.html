Sport Automatic Transmission (2TB) - Complete VO Coding
Popular Feature: Faster, more aggressive gear shifts with enhanced throttle response. This is the FULL retrofit including Vehicle Order modification.
‚ö†Ô∏è IMPORTANT: This procedure requires Windows PC with ESys. Mobile apps (Android/iOS) cannot perform VO/FA coding. Standard automatic transmission code is 205. Sport Automatic Transmission code is 2TB.

üìã Prerequisites

  • Windows PC with ESys 3.38.2 or BimmerUtility
  • ENET cable connection
  • Paddle shifters on steering wheel (M Sport wheel or retrofit paddles)
  • Full battery charge or battery maintainer connected

üîß Step-by-Step VO Coding Procedure

1Connect to Vehicle

  • Enable diagnostic mode (press start button 3x, foot off brake)
  • Open ESys and click the Connect icon at top
  • Select correct TargetSelector for G20 (e.g., G20, not G20_DIRECT)
  • Select "Connection via VIN" and press "Connect"
  • Click OK when connection is established

2Read Current Vehicle Order (FA)

  • Select "Expert Mode" in ESys
  • Click on the "Coding" tab
  • In Vehicle Order box, click "Read" button to display car's FA
  • Click "Save" button and save the original FA (e.g., "G20_FA_Original.xml")
  • IMPORTANT: Keep this backup file safe for restoration

3Edit Vehicle Order

  • Click "Edit" button - view changes to FA-editor automatically
  • Expand FAList folder system on left until SALAPA-Element is visible
  • Click on SALAPA-Element to select it
  • View SALAPA-Elements in bottom window
  • Delete option code 205 (Standard Automatic Transmission)
  • Add option code 2TB (Sport Automatic Transmission)
  • Syntax: alpha-numeric with commas only, no spaces, in alphabetical order
  • Click "Save Changes" icon at upper right corner of lower window

4Calculate & Verify FA

  • Right-click on FA and select "Calculate FP"
  • If FA has errors, an error popup will appear - fix before continuing
  • After FP is calculated WITHOUT ERRORS, save the new FA
  • Save file with descriptive name (e.g., "G20_FA_2TB_Modified.xml")

5Write FA to Vehicle

  • Select "Expert Mode" (if not already selected)
  • Click on "VCM" tab
  • Select "VCM Master" tab
  • Click "Write FA FP" button to write new FA to car
  • Wait for process to complete

6Activate FA & Code Modules

  • Return to "Coding" tab in Expert Mode
  • Click "Read" to read the modified FA/VO from vehicle
  • Click "Activate FA (VO)" button - this is critical!
  • Read SVT from vehicle
  • VO Code the following modules:
    • BDC_BODY2 (or BDC2_Body)
    • EGS (Electronic Gearbox System)
    • DSC3 (Dynamic Stability Control)
    • DKOMBI2 (Instrument Cluster)
  • For each module: Right-click on ECU ‚Üí Select "CODE"
  • Wait for each module to complete coding

7FDL Coding (Module Parameters)

BDC_BODY (or BDC2_Body)
  • Navigate to PADDLES_VERBAUT
  • Change from "nicht_aktiv" to "aktiv"
  • Note: If you don't have paddle shifters, set to "nicht_aktiv" (feature works without paddles)
EGS (Electronic Gearbox System)
  • Navigate to Sport T/M or Sportschalter
  • Change from "disabled" to "enabled"
  • Or change to "aktiv [01]" depending on your software version
‚úÖ Results After Coding:
  • Instrument cluster displays S1, S2, S3 gears in Sport mode
  • Faster, more aggressive gear shifts at higher RPMs
  • Enhanced throttle response
  • More pronounced downshifts
  • Paddle shifters become functional (if equipped)
  • Launch Control compatibility enabled (requires separate LC coding)
üí° Alternative: Simple Coding (Without VO Modification)
If you already have paddle shifters and just want to enable sport transmission behavior without modifying the VO:
  • Use BimmerCode app (any platform) ‚Üí Transmission module ‚Üí Enable "Sport Automatic Transmission"
  • Or use BimmerUtility/ESys FDL coding in BDC_BODY and EGS modules as described in Step 7
  • This method doesn't add 2TB to your vehicle order but provides similar functionality
‚ö†Ô∏è Critical Warnings:
  • DO NOT right-click and select CODE on all ECUs at once - this will recode your entire car
  • Only code the specific modules listed (BDC_BODY2, EGS, DSC3, DKOMBI2)
  • Always save original FA backup before modification
  • Modifying VO/FA alone does nothing - you MUST code affected modules
  • Ensure battery is fully charged throughout the process
  • Do not disconnect power during coding process
Launch Control (Advanced - ESys Required)
Advanced Feature: Requires ESys/BimmerUtility on Windows. Cannot be coded with BimmerCode mobile apps.

Prerequisites:

  • Sport Automatic Transmission (2TB) must be coded first
  • Vehicle must have appropriate hardware support
  • Check iDrive help menu for "Launch" keyword to verify compatibility
ESys / BimmerUtility (Windows Only)
  • Complete the 2TB VO coding procedure above
  • Ensure option code 2TB is in vehicle FA
  • Code EGS (Electronic Gearbox System) module
  • Drive vehicle for 6 miles (10km) to initialize system
How to Use Launch Control:
  1. Put car in Sport mode (NOT Sport+)
  2. Shift to Manual mode (push shifter left)
  3. Enable DSC Sport or MDM mode
  4. Press brake pedal fully to floor
  5. Press accelerator pedal fully to floor
  6. Launch control flags appear on instrument cluster
  7. Release brake to launch
Note: May require 6 miles of driving between launches. Puts significant strain on drivetrain.

üìö Important Information & Resources

‚ö†Ô∏è Backup & Restore: BimmerUtility/BimmerCode automatically creates backups before coding. Keep external backups of NCD files and FA/VO files, especially before BMW software updates. All coding is performed at your own risk.
üîÑ After BMW Updates: Software updates (OTA or dealer) may reset coded features. Keep a list of applied codes and recode after updates. VO/FA modifications typically persist through updates but FDL coding may reset.

Compatibility Notes

  • Hardware Required: Features only work if necessary hardware modules are installed in your vehicle
  • Software Version: Some features require specific iStep/software versions
  • Regional Restrictions: ADAS features may have region locks. Functionality depends on local infrastructure and navigation map data
  • VIP Module: May require vehicle in diagnostic mode, engine OFF, and multiple coding attempts (3-4 tries)
  • Module Limitations: Secure Coding 2.0 control units cannot be accessed by any current FDL coding tools
  • Paddle Shifters: M Sport steering wheel recommended for 2TB coding. Standard sport wheel doesn't support clean paddle retrofit
